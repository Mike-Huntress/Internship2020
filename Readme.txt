Bond Trading System
Kenneth Shinn

The contents of the project are in the Project folder, and it consists of 3 function libraries and 3 Python scripts:

The 3 function libraries are:
indicators.py - get the indicators
signals.py - calculate and visualize the signals
returns.py - calculate the visualize the returns

The 3 Python scripts are:
main_one_country.py - file that puts together single-country trading system
main_multiple_countries.py - file that puts together the multi-country trading system
indicator_values_plot.py - visualize indicator values for US, CAN, AUS

Jupyter notebooks are in 'jupyter_notebooks' directory in the main directory.
'old_code' folder is just old scratch work. This can be ignored.
